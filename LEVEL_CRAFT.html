LEVEL CRAFT


  • Introduction

Level Craft is a deep learning-based project aimed at generating dungeons and room decorations for games. The project involves the design, development, and evaluation of deep neural networks (DNNs) using Keras, integrated with TensorFlow, to automate the creation of intricate game levels. The project culminates in an interactive application built with Unity, showcasing the practical application of AI in game development.

  • Project Title: Level Craft
  • Role: AI Programmer / UI Artist
  • Team Size: 3 Programmers
  • Platform: PC
  • Technologies Used: Unity, REST API, Flask, Keras, TensorFlow

Aims and Objectives

The primary goal of Level Craft was to develop a system capable of procedurally generating dynamic game levels, with a focus on dungeons and room layouts. The objectives were:

  • Loading and preprocessing datasets for training.
  • Designing and training DNN models with Keras for dungeon and room generation.
  • Evaluating the performance of these models.
  • Developing a Unity-based application to visualize the generated levels, integrating Flask for backend operations.

Analysis and Development

    Several DNN models, including Generative Adversarial Networks (GANs) and Variational Autoencoders (VAEs), were designed and trained using Keras within TensorFlow. Dungeon generation models utilized PixelCNN and GANs, while room generation employed VAEs and GANs. Each model was rigorously trained and tested, with the best-performing models selected for integration into the final application. Training methodologies included the use of data augmentation to improve model generalization. The training process involved optimizing hyperparameters such as learning rates and loss functions to ensure the best possible performance. The models were evaluated on their ability to generate high-quality dungeons and rooms that met specific design criteria.

Interactive Application and GUI

    The interactive application was developed using Unity, with a REST API backend built using Flask. This setup allowed seamless integration of AI-generated content into the Unity environment. The application enables users to select different dungeon and room models, generate new levels, and view the results directly in Unity. The combination of Flask and Unity provided a robust platform for real-time interaction with the AI models.

My Contributions

I contributed to the training process, focusing on data augmentation techniques to enhance room generation. I experimented with various models, including Conditional GANs for dungeon creation, and trained room models using PixelCNN and GANs. Additionally, I played a key role in evaluating the models, helping to develop metrics for assessing their performance. I also made significant contributions to the design and UI of the Unity application, using Adobe Illustrator to create UI elements that enhanced the overall user experience. My work on the Flask backend ensured smooth communication between the AI models and the Unity interface, making the application both functional and user-friendly.
